Face Recognition Attendance System

This project is a Face Recognition Attendance System built using Python and OpenCV. It leverages a CNN-based face detection model to locate faces in real time, and a trained model to identify who the person is. Once a person is recognized, their name and login time are recorded in a text-based attendance sheet.

🧠 Key Features:
  • ✅ Face detection using a Convolutional Neural Network (CNN)
  • ✅ Face recognition trained from a dataset of user images
  • ✅ Real-time recognition via webcam or video feed
  • ✅ Automatic attendance logging in a `.txt` file
  • ✅ Built with Python, OpenCV, and optional support for deep learning frameworks
